🚗 AutoFactory – Sistema de Gestión para Industria Automotriz
📖 Descripción del Proyecto

AutoFactory es un sistema de gestión orientado a la industria automotriz cuyo objetivo es administrar y controlar los procesos de producción y el manejo de componentes e inventarios dentro de una fábrica de vehículos.

El sistema permite coordinar las líneas de producción con los recursos necesarios, garantizando que los materiales, empleados y órdenes de producción estén correctamente integrados para lograr un ensamblaje eficiente.

La solución está organizada en módulos independientes pero conectados entre sí, permitiendo demostrar fácilmente la integración entre producción e inventario.

🎯 Objetivos del Sistema
Gestionar líneas de producción de vehículos.
Controlar el uso de componentes durante el ensamblaje.
Administrar inventario y materiales necesarios.
Integrar proveedores dentro del flujo del sistema.
Garantizar la disponibilidad de recursos para la producción.

🧩 Estructura de Módulos

⚙️ MÓDULO 1: Gestión de Producción (Líneas de Producción)

Este módulo se encarga de administrar todo el proceso productivo dentro de la fábrica.

Funciones principales:

Conecta vehículos con las líneas de producción.
Usa componentes necesarios para el ensamblaje.
Involucra empleados en el proceso productivo.
Maneja órdenes de producción.
Controla el proceso de ensamblaje.

Responsabilidad principal:
Coordinar la fabricación de vehículos asegurando que cada etapa del proceso productivo se ejecute correctamente.

📦 MÓDULO 2: Gestión de Componentes e Inventario

Este módulo administra los recursos físicos necesarios para la producción.

Funciones principales:

Controla los materiales necesarios.
Incluye gestión de proveedores.
Administra el inventario disponible.
Afecta directamente la producción según disponibilidad.
Permite demostrar la integración entre inventario y producción.

Responsabilidad principal:
Garantizar que los componentes y materiales estén disponibles cuando la producción los requiera.

🔗 Integración entre Módulos

El sistema integra ambos módulos mediante los componentes, ya que:

Producción utiliza componentes.
Inventario controla su disponibilidad.
La falta de materiales impacta directamente las órdenes de producción.

Esto permite simular un entorno real de fábrica automotriz.

▶️ Instrucciones de Uso
- Registrar los componentes y materiales en el módulo de inventario.
- Asociar proveedores a los materiales disponibles.
- Crear o gestionar el inventario existente.
- Definir vehículos y órdenes de producción.
- Asignar empleados y líneas de producción.
- Ejecutar el proceso de ensamblaje utilizando los componentes disponibles.
